# 📱 Time Converter App (My First Android Project)

🚀 This is my first Android application built using Kotlin + XML (Views system) in Android Studio.  
It is a simple beginner-friendly app that converts seconds into days, hours, minutes, and seconds.

---

# 🧠 What this app does?

This app takes input in seconds and converts it into a structured time format.

### Example:
Input: 90061 seconds  
Output:
1 Day  
1 Hour  
1 Minute  
1 Second  

---

# ⚙️ Features

- ⏱ Converts seconds into days, hours, minutes, seconds
- 📌 Input validation (empty input handling)
- 🎯 Simple and clean UI
- 📱 Beginner-friendly Android project
